## Run

untuk mencoba bubblenya, install semua yang dibutuhkan di `requirements.txt` baru jalankan `run.py`

Cara install `requirements.txt`
```
pip install -r requirements.txt
```

Cara menjalankan `run.py`
```
python run.py
```
